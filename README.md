# Usage

## Docker 

- https://registry.hub.docker.com/u/anarcher/go-cron/

```
$docker run anarcher/go-cron -h                                                                                                  2 ↵
Usage of go-cron:
    -cpu=4: maximum number of CPUs
    -file="crontab": crontab file path
```
